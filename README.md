# OFFORKA OS v5.4 — Pages Bundle

## Structure
- `index.html` uses standalone files:
  - `css/home.css`
  - `js/home.js`

- All other pages use shared files:
  - `css/style.css`
  - `js/main.js`

This means homepage changes will not break the inner pages.

## Included inner pages
- about.html
- speaking.html
- consulting.html
- media.html
- speakout.html
- booking.html
- contact.html
